# xiaoliApi 项目分析

## 1. 项目概述

**项目类型：** .NET 8.0 ASP.NET Core Web API  
**架构框架：** Volo.Abp (ABP Framework v8.3.4)，领域驱动设计（DDD）  
**解决方案：** xiaoliApi.sln（共 7 个项目）

## 2. 技术栈

| 技术 | 用途 |
|------|------|
| .NET 8.0 | 运行时框架 |
| Volo.Abp | 模块化应用框架 |
| Autofac | 依赖注入容器 |
| SqlSugar | ORM（PostgreSQL 数据库） |
| JWT Bearer | 身份认证 |
| Swagger/OpenAPI | API 文档 |
| FluentValidation | 数据验证 |
| AutoMapper | DTO 映射 |

## 3. 项目结构

```
xiaoliApi/
├── xiaoliApi.sln                    # 解决方案文件
├── init.sql                         # 数据库初始化脚本
├── xiaoli.Web/                      # Web 主机（入口）
├── xiaoli.Application/              # 核心应用层
├── xiaoli.Model/                    # 核心领域模型
├── xiaoli.Repository/               # 数据访问层
├── xiaoli.Shared/                   # 共享工具/帮助类
└── Permission/                      # 权限模块
    ├── xiaoli.Permission.Application/
    └── xiaoli.Permission.Model/
```

## 4. 分层架构

1. **Web 层** (`xiaoli.Web`) - API 控制器、中间件、启动配置、ABP 模块注册
2. **应用层** (`xiaoli.Application`) - 业务逻辑、服务、DTO、验证器、拦截器
3. **领域层** (`xiaoli.Model`) - 实体类、领域模型（SqlSugar 特性映射）
4. **仓储层** (`xiaoli.Repository`) - 数据访问、SqlSugar 配置、通用仓储基类
5. **共享层** (`xiaoli.Shared`) - 通用工具类、扩展方法
6. **权限模块** (`xiaoli.Permission.*`) - 用户账号、角色、菜单、权限管理（ABP 模块化）

## 5. 数据库

## 6. 启动流程

1. 创建 WebApplication Builder
2. 配置 Autofac 依赖注入容器
3. 加载 HisWebModule（依赖 PermissionApplicationModule）
4. 注册控制器、Swagger、JWT 认证等服务
5. 配置 SqlSugar 连接 PostgreSQL
6. 初始化 ABP 应用
7. 配置中间件管道（认证、授权、Swagger、路由）
8. 启动 Web 服务

## 7. 运行端口

- HTTP: `http://localhost:8096`
- HTTPS: `https://localhost:8097`
- Swagger: `http://localhost:8096/swagger/index.html`

## 10. 权限模块 API

| 服务 | 端点 | 说明 |
|------|------|------|
| SysUser | Create/Update/Delete/GetById/GetList | 用户 CRUD |
| SysUser | AssignRoles | 为用户分配角色 |
| SysUser | GetUserRoles | 获取用户角色列表 |
| SysRole | Create/Update/Delete/GetById/GetList | 角色 CRUD |
| SysRole | AssignMenus | 为角色分配菜单 |
| SysRole | GetRoleMenus | 获取角色菜单列表 |
| SysMenu | Create/Update/Delete/GetById/GetList | 菜单 CRUD |
| SysMenu | GetMenuTree | 获取树形菜单 |
| SysPermission | Create/Update/Delete/GetById/GetList | 权限 CRUD |
